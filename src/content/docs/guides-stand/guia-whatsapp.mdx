## O Momento Certo para Compartilhar

### Não Compartilhe Imediatamente

Um erro comum é enviar o link logo no primeiro contato. Resista a essa tentação! O momento ideal para compartilhar é quando você já:

- Entendeu as necessidades do cliente
- Criou uma conexão inicial
- Despertou curiosidade sobre o imóvel
- Qualificou minimamente o interesse

<Aside type="tip">
**Dica de ouro**: Antes de enviar o link, envie uma mensagem como: "Encontrei um imóvel que tem tudo a ver com o que você me falou. Posso te mostrar?" e aguarde a confirmação.
</Aside>

## Técnicas de Engajamento

### Use Mensagens em Camadas

Não despeje todas as informações de uma vez. Crie uma conversa:

**Mensagem 1** (Preparação):
```
João, lembrei de você quando vi esse LANÇAMENTO hoje! 
Tem exatamente aquele estilo moderno que você gosta.
```

**Aguarde resposta**

**Mensagem 2** (Compartilhamento):
```
Olha só esse empreendimento. No Stand Digital você 
pode ver o decorado em 360°, a vista da varanda e 
todos os detalhes!

Repara especialmente na cozinha integrada e no 
pé direito alto 👇
[Link do StandDigital]
```

**Mensagem 3** (Após um tempo):
```
Conseguiu ver o tour virtual? O que achou da planta?
```

### Desperte Curiosidade com Escassez (Honesta)

<Aside type="caution">
Use apenas informações verdadeiras. Nunca minta sobre disponibilidade de unidades ou interesse de outros clientes.
</Aside>

Quando for verdade, você pode mencionar:
- "Esse lançamento acabou de abrir pré-venda HOJE"
- "Já temos 2 propostas em análise para unidades similares"
- "As primeiras 20 unidades têm desconto especial até o fim do mês"
- "Esse é um lançamento exclusivo, ainda não divulgado publicamente"

## O Poder do Áudio

### Quando Usar Áudio

Mensagens de áudio criam conexão e podem ser muito eficazes quando:

- Você quer passar entusiasmo genuíno pelo imóvel
- Precisa explicar algo mais complexo (localização, diferenciais)
- Quer criar intimidade e confiança

**Estrutura de um bom áudio:**
1. Cumprimente pelo nome (3 segundos)
2. Relembre a conversa anterior (5 segundos)
3. Apresente o imóvel com entusiasmo (15-20 segundos)
4. Faça uma pergunta ou call-to-action (5 segundos)

**Total: 30-35 segundos máximo**

<Aside type="note">
Áudios longos raramente são ouvidos até o fim. Seja objetivo e energético!
</Aside>

## Erros Fatais a Evitar

### ❌ Spam de Imóveis

Enviar 5, 10, 15 links de uma vez é o melhor jeito de ser ignorado. O cliente vai:
- Se sentir sobrecarregado
- Não olhar nenhum com atenção
- Achar que você não entendeu o que ele quer

**✅ Solução:** Envie 1 ou no máximo 2 imóveis por vez, bem selecionados.

### ❌ Compartilhar e Sumir

Enviar o link e não dar continuidade é perder a oportunidade.

**✅ Solução:** Sempre faça follow-up:
- Aguarde 2-3 horas
- Envie uma mensagem leve: "Conseguiu dar uma olhada?"
- Se não responder, espere 1 dia e tente novamente com outro ângulo

### ❌ Ignorar Objeções

Cliente: "Achei caro"
Corretor ruim: "Mas está no preço de mercado!"

**✅ Resposta melhor:**
```
Entendo sua preocupação com o valor. 

Me conta, o que você achou do imóvel em si? 
Atende o que você precisa?

[Se sim] Ótimo! Vamos então ver o que podemos 
negociar com o proprietário...
```

## Horários Estratégicos

### Melhores Horários para Compartilhar

**Durante a semana:**
- **8h-9h**: Pessoas no transporte, checando o celular
- **12h-13h**: Horário de almoço, momento de relaxamento
- **18h-20h**: Após o trabalho, momento de planejamento pessoal

**Final de semana:**
- **Sábado 10h-12h**: Momento de planejar o dia, boa disposição
- **Domingo à noite**: Evite! Baixo engajamento

<Aside type="tip">
Adapte esses horários ao perfil do seu cliente. Um empresário pode preferir receber depois das 20h, enquanto um professor pode ter mais tempo durante o dia.
</Aside>

## Acompanhamento Estratégico

### Sequência de Follow-up Ideal

**Dia 0** - Compartilhamento inicial com contextualização

**Dia 0 (3 horas depois)** - Primeira checagem:
```
E aí, conseguiu dar uma olhada? 😊
```

**Dia 1** - Se não respondeu, agregue valor:
```
Vi que esse imóvel tem uma vantagem que não 
tinha mencionado: o condomínio inclui gás 
central. Isso economiza bastante no fim do mês!
```

**Dia 2** - Crie um senso de urgência (se verdadeiro):
```
Oi! Só avisando que recebi mais consultas sobre 
esse apartamento. Ainda está interessado em visitar?
```

**Dia 4** - Última tentativa, seja direto:
```
João, percebi que esse imóvel pode não ser 
exatamente o que você procura. 

Me ajuda a entender melhor: o que não te 
agradou? Assim consigo buscar algo mais 
alinhado com o que você quer 🙂
```

## Fechamento da Conversa

### Sempre Termine com uma Ação

Nunca termine uma mensagem sem indicar o próximo passo:

**❌ Ruim:**
```
Aqui está o link do empreendimento!
[Link]
```

**✅ Bom:**
```
Aqui está o empreendimento que comentei!
[Link do StandDigital]

Explora o tour 360° com calma e me avisa: 
você quer visitar o decorado amanhã 
ou sábado é melhor pra você?
```

<Aside type="tip">
Note que a pergunta final oferece duas opções de ação, não uma escolha entre "sim" ou "não". Isso aumenta significativamente as chances de resposta positiva.
</Aside>

## Checklist Antes de Enviar

Antes de compartilhar qualquer empreendimento, pergunte-se:

- [ ] Entendi realmente o que o cliente procura?
- [ ] Esse empreendimento atende as necessidades dele?
- [ ] Preparei o cliente antes de enviar o link?
- [ ] Destaquei o diferencial relevante para ELE?
- [ ] Mencionei os recursos visuais do Stand (360°, vídeos, etc)?
- [ ] Indiquei qual conteúdo ver primeiro no Stand?
- [ ] Terminei com uma pergunta ou call-to-action clara?
- [ ] Estou disponível para responder dúvidas nas próximas horas?
