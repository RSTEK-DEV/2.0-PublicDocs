A área de **Administração de Usuários** fica dentro de **Configurações** no painel do Stand Digital. Ela concentra o controle de acesso ao sistema e permite manter a operação organizada por perfil, empresa e responsabilidade.

## Onde encontrar

1. Faça login no sistema.
2. Escolha a empresa desejada.
3. Acesse o **Painel Principal**.
4. Entre em **Configurações**.
5. Abra **Administração de Usuários**.

## O que pode ser feito

Nesta área, usuários com permissão administrativa podem:

- cadastrar novos usuários;
- editar dados de usuários existentes;
- ativar ou desativar acessos;
- definir perfis e permissões;
- vincular usuários à empresa correta;
- revisar quem pode acessar cada módulo do painel.

## Boas práticas

- mantenha apenas usuários ativos com acesso liberado;
- revise permissões sempre que houver mudança de função;
- padronize perfis para evitar acessos excessivos;
- use a vinculação correta de empresa para impedir operações no ambiente errado.

<Aside type="note">
O conjunto de opções visíveis pode variar conforme o perfil do usuário autenticado. Nem todo usuário verá os mesmos recursos administrativos.
</Aside>

## Benefícios da gestão centralizada

Ao administrar usuários dentro de Configurações, a empresa garante:

- mais segurança no acesso ao sistema;
- melhor rastreabilidade de ações;
- organização por equipe, função e empresa;
- menor risco de permissões inconsistentes.
