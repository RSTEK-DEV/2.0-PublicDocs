O acesso ao Stand Digital segue um fluxo simples e controlado para garantir segurança, organização e separação correta entre empresas e operações.

## Fluxo de acesso

1. O usuário abre a **tela de login**.
2. Informa suas credenciais de acesso.
3. Após autenticar, seleciona a **empresa** que deseja acessar.
4. O sistema direciona o usuário para o **painel principal**.

<Aside type="tip">
Se o usuário tiver vínculo com mais de uma empresa, a etapa de seleção garante que ele entre no ambiente correto antes de iniciar qualquer operação.
</Aside>

## Tela de login

Na tela inicial, o usuário informa os dados necessários para autenticação. Esse passo valida o perfil e libera apenas os recursos compatíveis com sua permissão.

## Seleção de empresa

Após o login, o sistema pode exibir a lista de empresas disponíveis para aquele usuário. A escolha da empresa define:

- quais dados serão exibidos no painel;
- quais empreendimentos e materiais estarão disponíveis;
- quais permissões administrativas serão aplicadas.

## Painel principal

Depois da seleção, o usuário acessa o painel principal. Esse painel funciona como ponto central da operação e reúne os atalhos para os módulos disponíveis no perfil.

Normalmente, o painel concentra recursos como:

- visualização geral da operação;
- acesso a configurações;
- navegação para módulos administrativos;
- acompanhamento de conteúdo, empreendimentos e usuários.

## Configurações

Dentro do painel, a área de **Configurações** reúne parâmetros de administração do ambiente. É nela que o usuário autorizado encontra recursos de controle operacional e gestão de acesso.

Entre essas opções está a área de **Administração de Usuários**, responsável pelo cadastro, edição e controle de permissões.

## Quando usar este fluxo

Esse fluxo é importante para:

- empresas com mais de uma operação;
- equipes que compartilham a mesma plataforma com perfis diferentes;
- cenários em que é necessário controlar acesso por empresa e por função.
