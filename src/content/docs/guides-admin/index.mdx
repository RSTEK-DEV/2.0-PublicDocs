Este guia ensina o **passo a passo prático** para operar a área administrativa do Stand Digital.  
Ideal para incorporadores e equipes de marketing que desejam configurar lançamentos com eficiência.

## Neste guia você vai aprender:
- como fazer login e escolher a empresa correta;
- como acessar o painel principal;
- como localizar a área de configurações;
- como administrar usuários e permissões;
- Como criar um novo lançamento.  
- Como adicionar materiais e plantas.  
- Como configurar as condições de venda.  
- Como convidar e gerenciar corretores.  
- Como visualizar relatórios e dados de engajamento.  

Cada seção traz instruções detalhadas e capturas de tela para facilitar o uso.
